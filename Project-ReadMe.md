# Portfolio Website

This project is a simple personal portfolio that showcases my work and provides an easy way to contact me. It is built with clean and responsive design principles.

# Features

- Responsive layout for mobile and desktop
- Sections for projects, skills, and about me
- Contact form / contact details
- Simple and clean design

# Technologies Used

- HTML
- CSS / Tailwind
- JavaScript (Vanilla)

# Contact

If you want to reach me, you can use the contact section on the website.
