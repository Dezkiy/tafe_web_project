WareLog

WareLog is a cross-platform warehouse tracking system built with a .NET MAUI mobile client and a Python FastAPI backend. The mobile app collects detailed information about incoming goods — including customer details, receiver and driver information, item descriptions, photos, and signatures — and sends it securely through RESTful API calls.

The backend, built with FastAPI, validates and processes the data before storing it in a centralized PostgreSQL database. This setup allows for fast, reliable, and scalable warehouse operations, while remaining easy to extend for future business requirements.
